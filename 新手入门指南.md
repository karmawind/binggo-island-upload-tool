# 新手入门指南 —— 从零开始使用 social-auto-upload

> 这份指南面向完全零基础的小白用户。只要你有一台 Windows 电脑，跟着步骤一步步来，就能成功运行这个软件。

---

## 第一部分：安装准备工作

### 1. 安装 Python（编程语言环境）

软件是用 Python 写的，你的电脑上需要有 Python 才能运行它。

1. 打开浏览器，访问 Python 官网下载页面：https://www.python.org/downloads/
2. 点击页面上的黄色按钮 **"Download Python 3.x.x"** 下载最新版本
3. 双击下载好的安装程序
4. **非常重要！** 安装界面的底部有一个选项 **"Add python.exe to PATH"**，**一定要勾选它！**（如果不勾选，后面会出错）
5. 点击 **"Install Now"** 等待安装完成
6. 安装完成后，点击 **"Close"** 关闭安装程序

**验证是否安装成功：**
1. 按键盘上的 `Win + R` 键，输入 `cmd`，按回车，打开命令提示符（一个黑底白字的窗口）
2. 输入以下命令并按回车：
   ```
   python --version
   ```
3. 如果显示类似 `Python 3.12.x` 的版本号，说明安装成功

---

### 2. 安装 Node.js（前端运行环境）

软件的前端界面需要 Node.js 来运行。

1. 打开浏览器，访问 Node.js 官网：https://nodejs.org/
2. 下载 **LTS（长期支持版）** 版本
3. 双击安装程序，一路点击 **"Next"** 直到安装完成
4. 不需要修改任何选项，保持默认即可

**验证是否安装成功：**
1. 打开命令提示符（`Win + R` → 输入 `cmd` → 回车）
2. 输入以下命令并按回车：
   ```
   node --version
   ```
3. 如果显示类似 `v20.x.x` 的版本号，说明安装成功

---

### 3. 获取项目代码

如果你已经拿到了项目文件夹（比如别人传给你的压缩包，解压到了桌面），可以跳过这一步。

如果你需要从 GitHub 获取代码：
1. 下载并安装 Git：https://git-scm.com/download/win（安装时保持默认选项）
2. 打开命令提示符，输入以下命令下载项目：
   ```
   cd Desktop
   git clone https://github.com/dreammis/social-auto-upload.git
   cd social-auto-upload
   ```

> **接下来所有操作都在项目文件夹中进行。** 假设你的项目文件夹路径是 `C:\Users\你的用户名\Desktop\social-auto-upload`。

---

### 4. 安装项目依赖

#### 4.1 安装 Python 依赖

1. 打开命令提示符
2. 进入项目文件夹（把下面的路径替换成你自己的实际路径）：
   ```
   cd C:\Users\你的用户名\Desktop\social-auto-upload
   ```
3. 创建虚拟环境（不用担心这是什么，照做就行）：
   ```
   python -m venv .venv
   ```
4. 激活虚拟环境：
   ```
   .venv\Scripts\activate
   ```
   激活成功后，你会看到命令行前面出现了 `(.venv)` 字样
5. 安装所有需要的 Python 包：
   ```
   pip install -e .
   ```
   这一步可能需要几分钟，会下载很多文件，耐心等待。看到最后出现 `Successfully installed...` 就说明成功了。
   > 如果上面的命令报错，也可以用 `pip install -r requirements.txt` 代替。

#### 4.2 安装浏览器自动化驱动

软件需要操控浏览器来自动发布内容，所以要安装一个浏览器驱动：

```
pip install patchright
python -m patchright install chromium
```

等待下载完成即可。

#### 4.3 初始化数据库

软件需要一个数据库来存储你的账号和文件信息。运行以下命令创建数据库：

```
python db/createTable.py
```

看到 `表创建成功` 就说明完成了。

#### 4.4 配置项目

项目有一个配置文件需要手动创建：

1. 在项目根目录找到 `conf.example.py` 文件
2. 把它**复制一份**，重命名为 `conf.py`
3. 一般情况下不需要修改里面的内容，保持默认即可

#### 4.5 安装前端依赖

1. 在命令提示符中进入前端文件夹：
   ```
   cd sau_frontend
   ```
2. 安装前端依赖包：
   ```
   npm install
   ```
   这一步也需要几分钟。等待完成即可。
3. 安装完成后，返回项目根目录：
   ```
   cd ..
   ```

## 第二部分：启动软件

**每次使用软件前，你都需要按以下步骤启动。** 需要打开 **两个** 命令提示符窗口。

### 步骤 1：启动后端服务器

1. 打开一个命令提示符窗口（`Win + R` → `cmd` → 回车）
2. 进入项目目录并激活虚拟环境：
   ```
   cd C:\Users\你的用户名\Desktop\social-auto-upload
   .venv\Scripts\activate
   ```
3. 启动后端：
   ```
   python sau_backend.py
   ```
4. 看到类似 `Running on http://0.0.0.0:5409` 的信息，说明后端启动成功
5. **注意：这个窗口不要关闭！** 后端在运行期间需要一直开着

### 步骤 2：启动前端界面

1. **再打开一个新的**命令提示符窗口（`Win + R` → `cmd` → 回车）
2. 进入前端目录并启动：
   ```
   cd C:\Users\你的用户名\Desktop\social-auto-upload\sau_frontend
   npm run dev
   ```
3. 看到类似以下信息说明启动成功：
   ```
   VITE v6.x.x ready in xxx ms
   ➜ Local: http://localhost:5173/
   ```

### 步骤 3：打开软件界面

1. 打开浏览器（推荐使用 Chrome 或 Edge）
2. 在地址栏输入：`http://localhost:5173`
3. 按回车，就能看到软件的界面了

**恭喜你，软件已经成功运行了！**

## 第三部分：登录你的社交媒体账号

在使用发布功能之前，你需要先登录你的各个平台账号。

### 通过网页界面登录（推荐）

1. 在软件界面上找到 **"账号管理"** 页面
2. 点击 **"添加账号"** 或 **"登录"** 按钮
3. 选择你要登录的平台（如抖音、小红书、B站等）
4. 输入一个账号名称（随便取一个，方便你自己区分，比如"我的抖音号1"）
5. 点击登录后，会自动弹出一个浏览器窗口
6. 在弹出的浏览器中用手机扫码或输入账号密码完成登录
7. 登录成功后，浏览器会自动关闭，软件界面上会显示账号状态为"有效"

## 第五部分：使用命令行（可选）

如果你习惯用命令行，也可以不通过网页界面，直接用命令来操作。

> 以下所有命令都需要先激活虚拟环境：
> ```
> cd C:\Users\你的用户名\Desktop\social-auto-upload
> .venv\Scripts\activate
> ```

### 登录账号

```bash
# 抖音登录
python sau_cli.py douyin login --account 我的抖音号

# 小红书登录
python sau_cli.py xiaohongshu login --account 我的小红书号

# 快手登录
python sau_cli.py kuaishou login --account 我的快手号

# 百家号登录（推荐加 --headed 看到浏览器界面）
python sau_cli.py baijiahao login --account 我的百家号 --headed

# 什么值得买登录
python sau_cli.py smzdm login --account 我的账号 --headed

# 头条号登录
python sau_cli.py toutiao login --account 我的头条号 --headed

# 携程登录
python sau_cli.py ctrip login --account 我的携程号 --headed
```

### 发布视频

```bash
# 发布视频到抖音
python sau_cli.py douyin upload-video --account 我的抖音号 --file 视频路径.mp4 --title "视频标题"

# 发布视频到小红书
python sau_cli.py xiaohongshu upload-video --account 我的小红书号 --file 视频路径.mp4 --title "视频标题"
```

### 发布图文

```bash

python sau_cli.py baijiahao upload-article --account 我的百家号 --title "文章标题" --content "文章正文" --images 图片1.png 图片2.png --headed

# 什么值得买发布图文
```

## 常见问题

### Q: 运行 `pip install -r requirements.txt` 报错怎么办？
**A:** 确保你已经激活了虚拟环境（命令行前面有 `(.venv)` 字样）。如果某个包安装失败，可以尝试先升级 pip：`pip install --upgrade pip`，然后重新运行安装命令。

### Q: 运行 `patchright install chromium` 报错怎么办？
**A:** 可能是网络问题。可以尝试多运行几次。如果还不行，检查你的网络连接。

### Q: 打开 `http://localhost:5173` 显示无法访问？
**A:** 检查两个服务是否都在运行：后端服务（端口 5409）和前端服务（端口 5173）。确认两个命令提示符窗口都还开着，没有报错退出。

### Q: 登录账号后显示登录失败？
**A:** 可能是你扫码太慢了，cookie 过期了。重新点击登录再试一次，这次快一点扫码。

### Q: 发布视频/文章失败？
**A:** 常见原因：
- 账号 cookie 过期了，需要重新登录
- 视频文件太大（限制 160MB）
- 图片格式不对（支持 jpg、png、gif、bmp、webp）
- 标题太长或包含不支持的特殊字符

### Q: 虚拟环境激活后退出命令提示符，下次还需要重新激活吗？
**A:** 是的，每次新打开命令提示符都需要重新激活虚拟环境（执行 `.venv\Scripts\activate`）。

### Q: 软件不用的时候需要做什么？
**A:** 直接关闭两个命令提示符窗口即可，后端和前端服务都会自动停止。
